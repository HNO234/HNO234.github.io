109-2 交大修課心得

Posted on 2021-06-20 Edited on 2021-07-14

這學期終於結束啦~ OwO
這個 blog 也放好久快發霉了，紀錄一下這學期修的課吧
喔對了，如果有人想參考我的 blog 選課的話，因為我的課表通常跟一般大一很不一樣（？），可以多去參考別人的

主科

資料結構與物件導向程式設計（詹力韋/資工系大一下必修/3 學分）

如同課名所示，就是 OOP+DS。據說有的班級亂搞到只教 DS 的，幸好這個班還算正常兩個都有教（？）
期中考以前教 OOP 部份，教一些 OOP 核心概念如封裝、繼承，然後用 C++ 實作那些概念。
有時候會扯一些跟 OOP 比較無關的 C++ 語言特性如指標、Namespace、Friend Function 等。
有些人可能會覺得不需要扯太多 C++ 語言特性，我是覺得還好啦，當作你所不知道的 C 語言來學也不錯。
只是期中考題目真的有點太刁鑽了 QQ，專門找一些 C++ 奇怪的寫法來考。全班平均 6X，幸好還有 85 以上。

DS 部份教了一些基礎資料結構（Stack、Queue、BST、AVL Tree、Graph 等等），順便帶一些圖論演算法如 DFS、BFS、MST、最短路。
大部份在高中時期都學過，就輕鬆非常多。這部份的考試題目也正常很多，基本上就是有念就有分。
不過有一點小小抱怨一下，他的圖論演算法複雜度都亂推阿 @@@，是很明顯就能找到錯的那種。

這學期有兩次大作業，OOP 跟 DS 各一個。
OOP 部份是要寫一個簡單的 Text-based RPG，我對寫遊戲沒啥興趣就算了（？）
DS 是實作排序、MST、最短路演算法加寫報告，花最多時間的果不其然就是寫報告 XDD

整體來說教的還算 OK 啦，只是要把超多內容塞到一個學期裡面教，所以隨時都在趕課就是。

離散數學（葉宗泰/資工系大一下必修/3 學分）

上課內容大概包含了邏輯、計數原理、集合論、圖論等等。整體難度偏水，最難的大概是生成函數跟線性遞迴。
上課從不點名，有自己做的講義，但是講課就是一直抄板書，而且跟講義內容完全不一樣。
而且這堂課是英文授課，但是老師英文似乎不太好，口齒也不太清晰。
有時候會點人上來回答問題，不過問的問題都偏水，所以好像也沒什麼人在理他就是。
話說雖然我跟某些人覺得難度偏水，考試題目難度非常不一。第一次期中考幾乎每個人都滿分（考卷還被人發現一堆錯），但是第二次期中考滿分 100 平均才 50 出頭。
有六次的作業，大概就是課本每個單元挑三至四題練習題，大部份練習題題目偏水。
整體來說教學品質算沒有非常好，不過老師好像也是第一次開這堂課，就算了吧。

數位電路設計（范倫達/資工系大一下必修/3 學分）

傳說中高職生會打爆高中生的科目（？）
上課內容首先提到 Boolean Algebra，然後提到各種邏輯閘做成的邏輯判斷電路。之後就是最重要的兩個元件 Latch 跟 Flip-Flop 及應用，最後提到一點 RAM 跟 ROM 的原理。整體來說算有點吃力，但是沒有到非常難。
這堂課其他班級要寫 Verilog，就這個老師會講 Verilog，但是完全沒有相關作業也完全不考。沒有作業，只有給推薦練習題而已。有三到四次的小考。
教授滿喜歡學生問問題的，也很喜歡跟學生互動。只是有時候會聊天聊到忘記顧進度。
講解方面倒是滿清晰的，可以把很複雜的觀念講的很清楚。
小考、期中、期末題目除了課本練習題以外，會考一些比較靈活的題目，但是基本上都是有念有分。
順帶一提，因為疫情的關係最後幾周是線上授課考試，這科的期末考抓作弊是抓最嚴的（

物理二（李威儀/校定共同必修/4 學分）

跟上學期一樣是線上授課，固定時間回來考試。
跟上學期很涼很甜的課完全不一樣，論上課內容是本學期我覺得最難的科目，甚至覺得比其他主科難度高上一倍。
上課內容的話，前半大概就是先提到一些近代物理的發現，以及量子力學概論。下半學期是談如何把量子力學概念應用到半導體身上，和介紹一些半導體元件的運作原理。
不過如果以考題來說，因為很多都是他給的練習題和考古題，整體來說難度不高。
不過我覺得他很多公式推導都滿唬爛的，不過畢竟是概論性質就沒有太追究這一塊。公式推導幾乎沒考過，幾乎都是很直接的說明題。
至於有沒有學到東西嘛，我覺得還是有一點啦，不過感覺對做硬體的比較有幫助（？）

數值方法（莊榮宏/資工系選修/3 學分）

這堂課主要是在講一些數學上無法完全解決的問題，如何用電腦運算算出近似值，例如說解非線性方程、各種函數的微積分。有時候會提到一些其他實用的東西像是用多項式逼近未知曲線、更快逼近函數的近似值等等。上課用老師自製的投影片，老師上課就是照講義講課，聲音非常的催眠 XDD
考試就是記那些方法來考試，會叫你手算很多算到會死的東西（可以用計算機）。會考證明題，證明題大概會佔整份考卷 30% 左右。
整體來說我是不太喜歡記那些方法啦，但是幸好老師題目也沒有非常刁鑽，只要記得那些演算法不要粗心算錯就會有分（對啦我就是很容易粗心算錯啦 QQ）
喔對了，這堂課一個很大的特色是會講很多你微積分/線代課學到的東西怎麼優化，所以非常吃微積分跟線代（尤其微積分），微積分被當的千萬不要修這堂課（（
之後可能會在這個 Blog 貼一些這堂課學到的東西 XD

競技程式設計一（謝旻錚/資工系選修/3 學分）

卡車開的課欸ww
其實我選這堂課八成只是為了符合學校的畢業需求（？），畢竟競程是我高中主業，我去修對我幫助不大？
我也都沒去上課，好像就是教 Greedy、圖論、DP 那些。兩次大考試也是沒什麼意外的都滿分（只是期末挑了一題 CF rating 2800 的題目，幸好我看過那題題解 XDD）。期中考難度大概就是基礎演算法，期末考難度大概到線段樹、BIT、位元 DP。

人工智慧概論（陳奕廷/資工系選修/旁聽）

因為開學時期的一些亂流所以就變旁聽了（？）
不過我也滿慶幸那時候是旁聽的，因為我後來發現 26 學分我應付不過來 XDD
因為是旁聽就不付修課心得啦，我後面都沒去上課（X）

其他科目

音樂概論（宋育任/通識/2 學分）

aka 古典音樂史概論，大部份時間都是在介紹一些著名音樂家和著名作品（大部份都是幾乎每個人都聽過那種）。
除了剛開學幾周會介紹人聲跟樂器以外不太會碰樂理。
偶爾會聊到一些作曲家或演奏家的小故事，跟老師之前讀音樂系的時候的趣事，當作去聽故事也不錯。
考試幾乎都是音樂題，就是放一段上課講過的音樂，問曲名跟作曲家。
期末會有一份報告是要去現場聽音樂會，但是因為疫情，這次老師特別開放聽線上音樂會。期末考因為疫情，改成另外一份報告。
整體來說算有趣啦，只是考試要稍微準備一下，因為有些曲目跟樂器真的聽起來很像 XD

進階英文聽力與寫作（郭詩芝/校定共同必修/2 學分）

跟 PTT 上的說法完全不同（？），這堂是超級硬的英文課，作業偏多考試也小難。
作業部份每周要預習一個單元，並且提出兩個值得思考的申論題 + 你的看法，單純的 Yes/No Question 或有標準答案的題目是會被打槍的。有時候可能會隨機點人上來分享你的申論題。
會有 6-7 次的寫作練習，可能在課堂上寫或當回家作業，每次字數 200-400 字不一而定。
考試部份，期中/Quiz（今年因為疫情沒有 Quiz）就是一般的英文考試+英聽，會從課本裡面出題，我沒預習所以爆慘 QQ
期末會丟很多時事影片，一部影片大概一個小時，叫你挑一部影片看完以後演講十分鐘。（今年因為疫情改丟演講影片）
不過感覺老師是真心想要培育學生，讓英文變成一輩子帶的走的能力。此外老師也很強調思辨能力，有時候會針對時事跟同學討論想法，像他就聊過大陸限鳳梨進口跟疫苗之亂 XDD
整體來說我覺得他是一個好老師啦，看起來有點嚴肅，其實對學生都滿 Nice 的。

服務學習一（張永儒/校定共同必修/0 學分）

學期一開始會有很多選項給你選，有打掃、安養院照護、資工生活影片製作、開源軟體推廣等等。
我是選校園 FAQ，簡單來說就是幫校園 bot 蒐集資料，每個禮拜交固定數量的資料上去。
基本上就是苦力活，很多資料都超級難找。可能有個好處是可以熟悉一個你之前完全不熟的科系，像我現在就熟管科系在幹嘛（X）
喔還有會叫你去聽兩場服學演講，大概就是請人分享一些志工服務的經驗。

生涯規劃及導師時間（孫春在/校定共同必修/0 學分）

本學期最討厭的課。
平心而論中間穿插的演講有些不錯，像之前有一堂課講說如何做一個好的 Presentation 我就覺得還不錯。
但是這學期除了演講以外還多了兩項意義不明的作業。其中一項是粉專經營，這個我還可以接受，畢竟推銷自我還是滿重要的。
但是另一個是用麥塊蓋理想校園，這個我就完全不知道意義在哪了，可能是想要推學生進麥塊坑（

藝文賞析教育（藝文中心/校定共同必修/0 學分）

跟上學期一樣是三展一演，只是今年因為疫情可以接受四展，且可以貼現上展覽，沒啥好說的。

大一體育（陳玫伶/校定共同必修/0 學分）

就是打打排球、丟丟壘球、玩玩羽球桌球，也沒什麼好說的。

整體心得

這學期因為必修本來就多（一般學生必修有微積分、物理、OOP、離散、數電、英文），再加上四個零學分的科目，這學期感覺特別累，有種時間被偷走的感覺。
在學期中的時候常常都是一邊抱怨「為什麼我要做這個垃圾作業」一邊含淚繼續做的。
不過中間還是有很多開心的部份，像是跟同儕一起修課，各種校外活動像是 YTP 專案、ICPC 團練、AIS3 等等。
希望這大學四年不要白費。
