Tricks

  • Ctrl + D 可以直接關掉現在的 terminal

  • Ctrl + L 或 clear 可以清除現在的 terminal 內容

  • Ctrl + A,E 是到一條指令的頭/尾

  • Alt + F,B 是往前或往後一的 word

  • Ctrl + R 是 search shell history

  • Ctrl + K 是剪下行尾

  • Ctrl + U 是剪下行首

  • Ctrl + Y 是貼上

  • Ctrl + W 是砍掉前面一個字

  • Ctrl + X + E 可以開 editor 編輯現有指令

  • Alt + . 可以重複上一指令的參數
