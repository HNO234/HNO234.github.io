lvalue & rvalue in C++

注意：以下內容很多都是我今天才知道的，所以內容可能很不精確，如果看到有錯的可以糾正我w

今天要介紹的主題是 lvalue 跟 rvalue。其實這是在 C 就有的東西了，只是在 modern C++ 有對這方面作大幅度的變化，對了解 modern C++ 滿有幫助的。

What is lvalue & rvalue

每個 C/C++ 當中的 Expression 都有 type 和 value 兩種屬性。lvalue 和 rvalue 是兩種 type 的種類，簡單來說：

  • lvalue: 實際佔據一塊記憶體的值，例如自己定義的變數、指標、reference 等等。
  • rvalue：不佔據記憶體，一離開該敘述就會消失的值，例如常量、暫存值等。

值得注意的是，在 C++ 11 以後還多出了三個分類，分別叫 glvalue、prvalue、xvalue，和上面的兩種傳統分類有重疊關係。他們之間的關係與詳細定義可以參考這裡。
