Chrome Extension 簡單教學

這個是我現在在寫的 side project 之一。預計的功能會是每天在 Extension 中選擇使用者自定義的語句之一。等開發完以後應該也會寫一篇開發筆記講一下實作細節。至於為什麼會先嘗試 Chrome，可能只是看起來比較簡單吧（X）

以下內容參考自官方教學。其他比較詳細的內容可以參考這裡。

Chrome Extension 的構造很簡單，底下所有東西包成一個資料夾就可以了。

mainfest.json

整個 Extension 最核心的部份。一個簡單的範例如下：

  • Name, Description, Version, icons: 就是字面上的意思。
  • Manifest Version: Chrome Extension 有分很多版，現在是第三版，所以填 3 就可以了。
  • Background: 跑在後台的 JavaScript worker。以前 (Manifets v2) 這個欄位是一個 script，可以選擇要 persistent 還是要 event，現在他可以自行判斷 (?) 需不需要。直接填上你寫好的 JavaScript worker 就好。
  • Permissions: 你想要 Chorme 給你什麼權限。完整的權限列表在這裡。
  • Action: 點擊每個頁面時會跳出哪個 HTML 頁面。

剩下的 HTML、CSS、JavaScript 應該就跟一般寫網頁差不多了，不過還是簡單帶一下一些重要的檔案。

popup.html

就是在你點 Chrome extension 圖示以後會跳出的頁面。

background.js

這裡只是簡單示範一下在每次 Extension 載入時會做的事。chrome.storage 是 Chrome 內部的一個存 user data 的地方。其他 API 可以參考這裡。

測試 Chrome Extension

如果需要測試寫好的 Chrome Extension 的話，直接用 Chrome 進入 chrome://extensions，打開 Developer Mode，之後再點 Load unpacked 點選你放以上檔案的資料夾即可。
